## Блок 1 Допустимые символы значения email-адреса
### Название тест-кейса: Email на латинице с цифрами
   **Шаги воспроизведения:**
     1.ввести в поле ввода email "test123@mail.ru"
     2.нажать кнопку "Проверить"
